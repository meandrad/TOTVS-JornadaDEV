
# Função ou Procedimento

1. `Str()`

   - É uma **função**, pois converte um valor numérico em uma string e retorna esse valor.
2. `QOut()`

   - É um **procedimento**, pois imprime informações na tela e não retorna nenhum valor.
3. `Date()`

   - É uma **função**, pois retorna a data atual.
4. `Len()`

   - É uma **função**, pois retorna o tamanho de uma string, array ou outro tipo compatível.
5. `Upper()`

   - É uma **função**, pois retorna a string convertida para letras maiúsculas.
6. `ClearScreen()`

   - É um **procedimento**, pois limpa a tela e não retorna nenhum valor.
